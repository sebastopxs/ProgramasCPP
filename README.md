# Prácticas de Programación en C++ 🚀

Este repositorio contiene códigos, ejercicios y soluciones desarrolladas durante el ciclo académico, enfocados en el dominio de estructuras de datos y algoritmos.

## 🛠️ Contenido
* **01_Matrices:** Operaciones con arreglos bidimensionales (Suma, multiplicación).
* **02_Vectores:** Uso de la librería `<vector>` y memoria dinámica.
